Cuenta de aspirante

Cada estudiante crea su usuario para registrar información personal y dar seguimiento a su admisión.

Enlaces compartidos

No se almacenan archivos en la plataforma. El estudiante solo registra la URL del documento compartido.

Revisión administrativa

El personal administrativo puede revisar enlaces, cambiar estados y dejar observaciones visibles al estudiante.

Requisitos principales

Documentos solicitados para admisión

El sistema permite dar seguimiento a los requisitos establecidos para el ingreso a programas de postgrado.

✓

✓

Créditos universitarios confrontados.

✓

✓

Cartas de solicitud de ingreso.

✓

✓

✓

Certificación de conocimiento del idioma inglés.

Aviso importante

La plataforma no almacena documentos

El aspirante debe compartir sus documentos mediante enlaces externos, como Google Drive, OneDrive, Dropbox u otra plataforma autorizada. Es responsabilidad del aspirante verificar que el enlace tenga permisos de visualización para que la administración pueda revisarlo.

Recomendación

Antes de enviar un enlace, abre la URL en una ventana privada para confirmar que se puede visualizar.
